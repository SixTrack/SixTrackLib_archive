Benchmarked Results
-------

This folder contains the benchmark results. The description of each files is as follows:

1. `errf_PyC`: comparison between Python `wofz()` and C version of `errf()` function.  
2. `errf_SixC`: comparison between FORTRAN version and C version of `errf()` function.
3. `maps_PyC`: comparison between Python and C versions of Drift, Multipole, Align and Block Maps.
4. `maps_SixC`: comparison between FORTRAN and C versions of Drift, Multipole, Align and Block Maps.
5. `wzsub_SixC`: comparison between FORTRAN version and C version of `wzsub()` function.
